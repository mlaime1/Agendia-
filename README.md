# Agendia (Expo)

Proyecto base de React Native usando Expo y TypeScript.

Quick start

1. Instalar dependencias:

```bash
npm install
# o
yarn install
```

2. Iniciar en modo desarrollo:

```bash
npm run start
# o
yarn start
```

3. Abrir en dispositivo/emulador usando Expo Go o los comandos:

```bash
npm run android
npm run ios
```

Notas
- Si no tienes el proyecto creado mediante `npx create-expo-app`, puedes crear uno nuevo con plantilla TypeScript y copiar estos archivos.
- Recomiendo usar `npx create-expo-app . --template expo-template-blank-typescript` si quieres que todo se configure automáticamente.
